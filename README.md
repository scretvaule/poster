# AAAI 2026 Poster - AdaFuse

This folder contains the LaTeX source code for the AdaFuse poster, based on the Gemini theme.

## Files

- `poster.tex`: Main LaTeX file.
- `beamerthemegemini.sty`: Gemini theme style file.
- `beamercolorthemenott.sty`: Color theme.
- `poster.bib`: Bibliography file (copied from paper).
- `figures/`: Contains figures used in the poster.
- `logos/`: Contains organization logos.

## Compilation Instructions (Overleaf)

1. Upload this entire folder (or zip) to Overleaf.
2. Open `poster.tex`.
3. Click on **Menu** (top left).
4. Set **Compiler** to **XeLaTeX**.
5. Click **Recompile**.

**Note:** This theme uses `fontspec` and requires specific fonts (Raleway, Lato) which are supported by XeLaTeX/LuaLaTeX. Do not use pdfLaTeX.
